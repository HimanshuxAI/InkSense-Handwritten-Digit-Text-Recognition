Digit Recognition Model

  • 99%+ test accuracy

Text Recognition Model

Based on githubharald/SimpleHTR

  • 5 CNN + 2 LSTM layers + CTC
  • Trained on IAM Handwriting Dataset
  • ~75% word accuracy, ~10% CER
  • Supports words and text lines
